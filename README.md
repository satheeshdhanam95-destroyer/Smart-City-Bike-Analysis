# Smart-City-Bike-Analysis
📌 Project Overview

This project analyzes bike sharing station data across different cities to understand station availability, bike usage, and operational performance.
The analysis was performed using Microsoft Excel and Power BI, where the dataset was cleaned, transformed, and visualized to generate meaningful insights about bike distribution and infrastructure.

The dashboard helps identify bike availability, station capacity, and operational status, enabling better decision-making for improving bike-sharing systems.

🎯 Objective

The main objective of this project is to:

Analyze bike sharing stations across different cities.

Understand bike availability and stand capacity.

Identify high-demand and low-utilization areas.

Visualize the geographical distribution of bike stations.

Evaluate the operational status of bike stations.

📂 Dataset

The dataset contains information related to bike sharing stations including:

City

Station Name

Position (Latitude & Longitude)

Available Bikes

Available Bike Stands

Total Bike Stands

Station Status

The data was cleaned and transformed before creating the dashboard.

🔄 Data Cleaning & Transformation

Data preparation was done using Power Query in Power BI.

Steps performed:

Removed unnecessary columns

Checked and corrected data types

Split Position column into Latitude and Longitude

Handled missing values

Converted dataset into a structured format suitable for analysis

📊 Power BI Dashboard Features

The dashboard includes the following visualizations:

1️⃣ Total Bikes by City

Shows the total number of bikes available in each city.

2️⃣ Total Bike Stands by City

Displays the station capacity available across different cities.

3️⃣ Available Bikes vs Bike Stands

Compares available bikes with available stands to understand supply-demand balance.

4️⃣ Bike Stand Status

Shows the distribution of stations based on their operational status.

5️⃣ Map Visualization

Displays bike stations geographically using latitude and longitude coordinates.

6️⃣ Available Bikes by Region

Helps understand which areas have higher bike availability.

7️⃣ Interactive Slicers

Users can filter the dashboard by:

City

Station status

Other attributes

📈 Key Insights

Cities with more bike stands tend to have higher bike availability, indicating better infrastructure.

Some cities show lower bike usage despite high availability, suggesting possible underutilization.

Most bike stations are operational, ensuring service availability for users.

Bike stations are mostly concentrated in major urban regions.

💡 Recommendations

Redistribute bikes from low-demand areas to high-demand locations.

Expand bike stations in high usage regions.

Regularly monitor station operational status.

Improve infrastructure in areas with high user demand.
